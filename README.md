# Micra 🎙️

Micra is a lightweight, privacy-focused macOS menu bar application that gives you complete control over your microphone with a single keystroke.

## Features

- **Global Hotkey:** Mute and unmute your microphone from anywhere with a customizable shortcut.
- **Push-to-Talk:** Only unmute your microphone while holding down your chosen hotkey.
- **Auto-Lock:** Automatically switches back to your preferred microphone if macOS changes the input device (e.g., when connecting AirPods).
- **Audio Feedback:** Optional sound cues when muting or unmuting.
- **Privacy First:** Micra operates entirely locally. No audio is recorded, processed, or transmitted.
- **Native Experience:** Built with SwiftUI for a modern, efficient, and seamless macOS experience.

## Installation

Visit [micr.app](https://micr.app) to download the latest version.

## Usage

1. Launch Micra.
2. Click the microphone icon in your menu bar to toggle mute manually.
3. Use the popover to:
   - Record a custom global shortcut.
   - Enable "Push-to-talk" mode.
   - Lock your input device.
   - Toggle audio feedback sounds.

## Requirements

- macOS 14.0 or later.
